اتبع الخطوات بالترتيب - ما يحتاج أي إعدادات معقدة

الخطوة 1: إنشاء المستودع

1 افتح الرابط التالي:
https://github.com/new
2 اكتب اسم المستودع: trading-platform
3 اختر Public
4 لا تضيف README ولا .gitignore ولا License
5 اضغط Create repository

الخطوة 2: رفع الملفات

بعد إنشاء المستودع، راح تطلع لك صفحة Empty repo.
اضغط على الرابط: "uploading an existing file"
